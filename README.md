## usage

1. clone project

```javascript

git clone https://github.com/ilz5753/RNCalendar.git
```

2. go to project directory

```javascript

cd RNCalendar
```

4. install dependencies

```javascript

yarn
````

5. run the project

```javascript

expo r -c
```
